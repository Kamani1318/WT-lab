JavaScript Boxes

alert()

The alert() method displays an alert box with a specified message and an OK button.

prompt()

The prompt() method displays a dialog box that prompts the visitor for input. It returns a string, or null if the user clicks "Cancel".

confirm()

The confirm() method displays a dialog box with a specified message and OK and Cancel buttons. It returns a boolean value, true if the user clicks "OK", and false if the user clicks "Cancel".
